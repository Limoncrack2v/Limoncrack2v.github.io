Introduction:

What is HTML?

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar páginas web. Permite organizar el contenido mediante etiquetas que definen elementos como títulos, párrafos, imágenes, enlaces y tablas. Además, sirve como base sobre la cual se integran otros lenguajes como CSS y JavaScript para diseñar y agregar interactividad a los sitios web.

Explain the head and the body tags

The <head> tag contains information about the webpage that is not directly shown to users, such as the title, metadata, links to stylesheets, and scripts.

The <body> tag contains all the visible content of the webpage, including text, images, buttons, and other elements that users interact with.

Basic Tags:

The <section> tag is used to group related content within a webpage, usually around the same topic. The <article> tag represents independent and complete content, such as a news story, blog post, or report. The <aside> tag is used for secondary or extra information, like side notes, advertisements, or related content. The <header> tag contains the introductory part of a page or section, such as the title, logo, or navigation menu. The <footer> tag represents the bottom part of a page or section and usually includes copyright information, contact details, or links.

Basic CSS

Explain:

A tag selector is used to apply styles to all elements of the same HTML tag, such as all paragraphs or headings. A class selector is used to style multiple elements that share the same class name, allowing reuse of styles across the page. A tag and class selector is more specific because it targets only a certain tag with a specific class. An id selector is used to style a single unique element on the page.

What is a class? What is an id? How are they related to CSS?

A class is an attribute that can be assigned to multiple elements to apply the same style to all of them. An id is an attribute that identifies a unique element, so it should only be used once per page. Both are related to CSS because they allow developers to target and style elements in a more precise way.

Explain the CSS Box Model

The CSS Box Model describes how every element on a webpage is represented as a rectangular box. This box includes the content, padding, border, and margin. The content is the actual information, the padding is the space around the content, the border surrounds the padding, and the margin is the space outside the element. Understanding this model helps control layout and spacing on a webpage.

Explain the media queries

Media queries are used to make webpages adapt to different screen sizes and devices. They allow the design and layout of a page to change depending on conditions such as screen width or device type. This makes websites more responsive and easier to use on phones, tablets, and computers.
